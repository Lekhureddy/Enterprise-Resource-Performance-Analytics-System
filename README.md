# Enterprise-Resource-Performance-Analytics-System

## Dataset
The dataset used in this project is a synthetic, industry-inspired project
management dataset designed to simulate real enterprise project scenarios.

- Size: 420 projects
- Domain: Engineering & resource management
- Includes missing values to reflect real-world data uncertainty
- Contains schedule, cost, resource utilization, and risk indicators

The raw dataset is stored in `data/raw/` and is not modified directly.
